PBS Baseline Pre-Roll 補點

產生時間：2026-05-26 10:29 +08:00 Asia/Taipei

本版修正切分後 PBS baseline 可能不足 200 點的問題，允許從切點左側穩定區補點。

PBS 分析

有切點時，PBS Segment 會從切點左側最多補 200 點作為 baseline pre-roll。

不足標記

若切點左側不足 200 點，sample 仍納入繪圖與統計，但會顯示實際點數 warning。

切點預覽

點選切點後，預覽圖會標出 PBS baseline pre-roll window，方便確認是否落在穩定區。

行為變更

  • process_single_file(..., segment="pbs") 在有 curve_split 時改用 raw[max(0, split_index - 200):]。
  • Drug apply search 會依實際 pre-roll 秒數往後平移，避免 drop 偵測提早。
  • Lanolin Segment 仍只使用 raw[:split_index]。
  • 不改 Excel，也不改既有 curve_splits metadata schema。
