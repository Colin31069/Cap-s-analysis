Legend 顯示修正

產生時間：2026-05-26 11:02 +08:00 Asia/Taipei

本版修正 Simple legend 顯示太多標記，以及 Matplotlib legend 中文狀態顯示成方框的問題。

行為變更

Simple legend 現在只顯示 N 1、N 2、N 3。Warning、inaccurate、manual drop 等標記只在 Detailed legend 顯示。

Detailed legend 的 warning 狀態改為 ASCII：[warning]、[inaccurate]，避免字型不支援中文時出現方框。

驗證

tests.test_plotting 已通過；完整驗證結果記錄於 CHANGELOG.md。
