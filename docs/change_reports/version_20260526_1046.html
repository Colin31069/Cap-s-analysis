手動 Drop 對齊

產生時間：2026-05-26 10:46 +08:00 Asia/Taipei

本版新增 plot-only 手動 drop alignment，用於修正單條曲線自動 drop 偵測偏移造成的視覺未對齊。

操作

先選 sample，再按 Adjust Drop，拖曳 highlighted handle 到真正 drop 點。

保存

手動值寫入 .skin_analysis_metadata.json 的 drop_time_overrides。

統計

修正只影響圖表顯示與匯出，不改 Delta %、Dixon Q、ANOVA 或 statistics CSV。

行為變更

  • 新增 drop_time_overrides metadata，依 file_name + segment 套用。
  • PBS Segment 的手動對齊不會影響 Lanolin Segment 或 Full Curve。
  • sample 清單與 legend 會標示 [manual drop]。
  • ProcessedSignal.drop_time 保留自動分析值；plotting payload 另外使用 display drop time。

測試重點

  • Metadata round-trip 新增手動 drop override。
  • Plotting 有 override 時才平移指定 sample。
  • Statistics 明確忽略手動 drop override。
