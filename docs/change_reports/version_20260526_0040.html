反應曲線切分與 HTML 報告

產生時間：2026-05-26 00:40 +08:00 Asia/Taipei

本版新增 lanolin/PBS 曲線切分，避免 lanolin 前段反應混入 PBS 分析；同時建立 HTML 變更報告流程。

切分

在 sample 清單選擇電極檔後，可開啟原始時間軸預覽並點選分界點。

PBS

有切點的檔案在 PBS 模式只分析右段，且右段時間從 0 秒重新開始。

Lanolin

Lanolin 模式只顯示已切分檔案的左段，用於檢查同濃度約 3 片電極反應曲線。

輸入與保存

  • 原始 Excel 檔不修改。
  • 切點保存於每個 experiment folder 的 .skin_analysis_metadata.json。
  • 新增欄位為 curve_splits，包含檔名、split_index、split_time_sec、左段標籤與右段標籤。
  • 舊 metadata 沒有 curve_splits 時仍可正常讀取，預設為空清單。

分析流程影響

模式 資料來源 時間軸規則
PBS Segment 有切點時使用右段；無切點時維持完整曲線 右段重新從 0 秒開始，再依既有 drop 偵測對齊繪圖
Lanolin Segment 只使用已切分檔案左段 維持原始起點 0 秒，不重算成 PBS 起點
Full Curve 完整原始曲線 沿用既有 drop 對齊顯示，主要作為檢查視圖
